Recreation API

Though this original repo uses a form of API that currently has no use, you can still use it with web requests.


Some forks of this repo might use databases or other methods of storing cameras and camera counts though and this original repo is also a simple recreation of the base of the website and will not contain full functionality.


GET requests

Get current camera count

GET [url]/req/cameraCount

Get total camera count

GET [url]/req/totalCameraCount
